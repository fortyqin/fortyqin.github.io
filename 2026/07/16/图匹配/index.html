定义

我们先来看一些关于图匹配的基础概念。

对于一个无向图 G=(V,E)G=(V,E)G=(V,E)：

  • 匹配：若边集 M⊆EM \subseteq EM⊆E 不包含自环，且其中任意两条不同的边 e1,e2∈Me_1, e_2 \in Me1​,e2​∈M 均没有公共顶点，形式化地，即对任意 e1=(u1,v1),e2=(u2,v2)∈M,e1≠e2  ⟹  {u1,v1}∩{u2,v2}=∅e_1 = (u_1, v_1), e_2 = (u_2, v_2) \in M, e_1 \neq e_2 \implies \{u_1, v_1\} \cap \{u_2, v_2\} = \emptysete1​=(u1​,v1​),e2​=(u2​,v2​)∈M,e1​=e2​⟹{u1​,v1​}∩{u2​,v2​}=∅，则称边集 MMM 是 GGG 的一组匹配。

下面，我们针对给定的图 G=(V,E)G=(V,E)G=(V,E) 及其匹配 MMM，进一步定义相关概念：

  • 匹配的大小：我们把集合 MMM 的基数 ∣M∣|M|∣M∣ 称为匹配 MMM 的大小。

  • 最大匹配：若匹配 MMM 的大小在图 GGG 的所有匹配中是最大的，即 ∣M∣=max⁡{∣M′∣:M′ 是 G 的匹配}|M| = \max \{ |M'| : M' \text{ 是 } G \text{ 的匹配} \}∣M∣=max{∣M′∣:M′ 是 G 的匹配}，则称 MMM 是 GGG 的一个最大匹配。特别注意，最大匹配可能不唯一，但它们的大小相同，且其大小上限为 ⌊∣V∣2⌋\lfloor \frac{|V|}{2} \rfloor⌊2∣V∣​⌋。
  • 最大权匹配：若匹配 MMM 的权值在所有匹配中是最大的，即 w(M)=max⁡{w(M′):M′ 是 G 的匹配}w(M) = \max \{ w(M') : M' \text{ 是 } G \text{ 的匹配} \}w(M)=max{w(M′):M′ 是 G 的匹配}，则称 MMM 是 GGG 的一个最大权匹配。显然，对于无权图（或边权全为 111 的图），最大权匹配即为最大匹配。
  • 完美匹配：若匹配 MMM 使得图中的所有顶点都是匹配点（即每一个顶点 v∈Vv \in Vv∈V 都关联 MMM 中的一条边），则称 MMM 为一个完美匹配。此时满足 ∣M∣=∣V∣2|M| = \frac{|V|}{2}∣M∣=2∣V∣​。完美匹配一定是最大匹配，且图的顶点数 ∣V∣|V|∣V∣ 必然为偶数。特别地，顶点数为偶数的完全图一定存在完美匹配。

为了解决匹配问题，我们需要引入交错路与增广路的概念。

对于图 G=(V,E)G=(V,E)G=(V,E) 及其匹配 MMM：

  • 交错路：图 GGG 中一条由匹配边且非匹配边交替出现的简单路径 P=(v1,v1,…,vk)P = (v_1, v_1, \dots, v_k)P=(v1​,v1​,…,vk​) 称为交错路。即对于任意 1<i<k1 < i < k1<i<k，边 (vi−1,vi)(v_{i-1}, v_i)(vi−1​,vi​) 与 (vi,vi+1)(v_i, v_{i+1})(vi​,vi+1​) 中有且仅有一条属于 MMM。
  • 增广路：若交错路 P=(v1,v1,…,vk)P = (v_1, v_1, \dots, v_k)P=(v1​,v1​,…,vk​) 的起点 v1v_1v1​ 和终点 vkv_kvk​ 均为非匹配点，则称 PPP 为一条增广路。
  • 增广路：对于一条增广路的边集 P⊆EP \subseteq EP⊆E，我们令 M←M⊕PM \gets M \oplus PM←M⊕P 称为增广。这里的 ⊕\oplus⊕ 表示对称差运算，即 M⊕P=(M∪P)∖(M∩P)M \oplus P = (M \cup P) \setminus (M \cap P)M⊕P=(M∪P)∖(M∩P)。

增广路

增广

我们可以先看一看增广的过程。

设 P=(v1,v1,…,vk)P = (v_1, v_1, \dots, v_k)P=(v1​,v1​,…,vk​) 是一条增广路。由于其起点 v1v_1v1​ 和终点 vkv_kvk​ 都是非匹配点，这意味着与 v1v_1v1​ 和 vkv_kvk​ 关联的路径首尾边必不属于 MMM，即第一条边 (v1,v2)∉M(v_1, v_2) \notin M(v1​,v2​)∈/M 且最后一条边 (vk−1,vk)∉M(v_{k-1}, v_k) \notin M(vk−1​,vk​)∈/M。

因为路径中匹配边与非匹配边交替出现，所以该路径必由非匹配边开始并以非匹配边结束。

由此可知，一条增广路中，非匹配边的数量比匹配边恰好大 111，即 ∣P∖M∣=∣P∩M∣+1|P \setminus M| = |P \cap M| + 1∣P∖M∣=∣P∩M∣+1。

当我们执行增广操作 M←M⊕PM \gets M \oplus PM←M⊕P 时，相当于将路径 PPP 中的匹配边与非匹配边状态翻转（原匹配边移出匹配，原非匹配边加入匹配）。增广后，新的匹配大小变为：$ |M \oplus P| = |M| - |P \cap M| + |P \setminus M| = |M| + 1 $。

这就完成了匹配大小增加 111 的增广过程。

值得关注的一个点是，我们一条增广路 P=(v1,v1,…,vk)P=(v_1,v_1,\ldots,v_k)P=(v1​,v1​,…,vk​) 只有起点和终点 v0,vkv_0, v_kv0​,vk​ 是非匹配点，中间的点都是匹配点，并且我们在增广后 v1,v2,…,vkv_1,v_2,\ldots,v_kv1​,v2​,…,vk​ 都是匹配点，这意味着每个点不会从匹配点变成非匹配点（在匹配过程中），我们的的匹配大小也可以直接通过匹配点的个数一半来描述。

我们现在已经知道，通过增广的方式，可以增大我们的匹配 MMM 的大小，但我们的目标可能为找到 GGG 的最大匹配，这样的方法是否奏效呢？

Berge 引理

Berge 引理：

对于图 G=(V,E)G=(V,E)G=(V,E) 和它的一个匹配 MMM，匹配 MMM 是最大匹配，当且仅当不存在相对于 MMM 的增广路。

证明：

首先必要性前文已经讨论过了，若 MMM 是最大匹配，那么一定就不存在相对于 MMM 的增广路。

另一方面，我们需要考虑，若不存在相对于 MMM 的增广路，那么 MMM 就是一个最大匹配。

反证法。假设存在一个比 MMM 更大的匹配 M′M'M′，这时候考虑对称差。构造一个图 G′=(V,M⊕M′)G'=(V,M\oplus M')G′=(V,M⊕M′)，对于每个点 v∈Vv \in Vv∈V，其度数只能是 0,1,20,1,20,1,2，这样的图 G′G'G′ 只能是单点、链、简单环的并。我们现在关注的是 M,M′M,M'M,M′ 的大小关系，单点和简单环中属于 M,M′M,M'M,M′ 的边是相等的，我们就只关注链。因为一组匹配中，两两边没有公共点，所以一条链的的边一定交错属于 M,M′M,M'M,M′，又因为我们假设 M′M'M′ 比 MMM 大，那么至少存在一条链 c=(v0,v1,…,vk)c=(v_0,v_1,\ldots,v_k)c=(v0​,v1​,…,vk​) 使得两条边 (v0,v1)∈M′,(vk−1,vk)∈M′(v_0,v_1) \in M',(v_{k-1},v_k) \in M'(v0​,v1​)∈M′,(vk−1​,vk​)∈M′，这条链就是一条相对于 MMM 的增广路，与假设矛盾，所以结论成立。

通过 Berge 引理，我们明白使用增广路改进匹配的方法是充分的，说明使用增广路求最大匹配在任意图上都是奏效的，但我们还没有设计出高效的算法以解决最大匹配问题。

为了落实求解最大匹配，我们可能还需要一个小结论，这将在介绍交错树时提出。

交错树

前面说了，我们改进匹配的一个充分方法就是增广，那我们就需要找增广路。交错树就是一个和增广路紧密相连的一个概念。

具体来说，我们从一个非匹配点 r∈Vr\in Vr∈V 出发，我们进行 BFS 或 DFS 找交错路，这就形成了一个交错树。形式化地，若子图 T=(V′,E′)⊆GT=(V',E') \subseteq GT=(V′,E′)⊆G 是一棵以 rrr 为根的树，且连接 rrr 到 v∈V′v\in V'v∈V′ 的任意一条路径都是交错路，且 TTT 是极大的，则称 TTT 是一个交错树。其中，树上深度为奇数的点称为奇点，树上深度为偶数的称为偶点。

那么如果令根的深度为 000，若以 rrr 为根交错树上存在了一个非匹配的奇点 v∈Vv\in Vv∈V，就相当于找到了一条 r→vr \to vr→v 的增广路，我们的算法流程就是进行 BFS 或 DFS 找增广路。

但我们可能会考虑：如果我们从一个点 rrr 开始进行 DFS 后若没有找到增广路，我们是否应当在找到其它增广路后再次从 rrr 开始增广。下面讨论了这一点。

结论：

若不存在相对于 MMM 的以 r∈Vr\in Vr∈V 作为起点的增广路，那么我们对 MMM 无论进行多少次增广后得到 M′M'M′，都依然不存在相对于 M′M'M′ 以 vvv 作为起点的增广路。

证明：

显然，我们只需要证明单次增广后不会新出现从 rrr 出发的增广路，且我们只需要考虑 rrr 为非匹配点的情况。

在 MMM 为当前的匹配时，我们构造出以 rrr 为根的交错树 T=(V′,E′)T=(V',E')T=(V′,E′)。我们将 TTT 中的顶点进行分类：记与 rrr 的距离为偶数（包括 000）的顶点集合为 Veven′V'_{even}Veven′​（即偶点），距离为奇数的顶点集合为 Vodd′V'_{odd}Vodd′​（即奇点），则 V′=Veven′∪Vodd′V' = V'_{even} \cup V'_{odd}V′=Veven′​∪Vodd′​。

由于不存在相对于 MMM 的以 rrr 出发的增广路，我们可以形式化地得出以下性质：

  1. TTT 中除根节点 rrr 外的所有顶点均为匹配点，即 V′∖{r}V'\setminus\{r\}V′∖{r} 中的顶点在 MMM 中都是匹配点，且它们的匹配边均属于 E′E'E′。
  2. 任何连接 V∖V′V\setminus V'V∖V′ 与 V′V'V′ 之间的边，其在 V′V'V′ 中的顶点必定属于奇点集合 Vodd′V'_{odd}Vodd′​（反之，若存在边 e=(u,v)e=(u, v)e=(u,v) 满足 u∈V∖V′u \in V \setminus V'u∈V∖V′ 且 v∈Veven′v \in V'_{even}v∈Veven′​，由于 uuu 在树外，边 eee 必为非匹配边，我们可以通过边 eee 将 uuu 作为奇点加入交错树，这与 TTT 的极大性矛盾）。

现在假设我们找到了一条用于本次增广的增广路 P=(v1,v2,…,vk)P=(v_1, v_2, \ldots,v_k)P=(v1​,v2​,…,vk​)。因为 PPP 的两个端点 v1,vkv_1, v_kv1​,vk​ 必须是非匹配点，而在 V′V'V′ 中唯一的非匹配点是根节点 rrr，且 PPP 的端点不可能是 rrr，所以必有 v1,vk∈V∖V′v_1, v_k \in V \setminus V'v1​,vk​∈V∖V′。

若 PPP 中存在边或顶点属于 V′V'V′：根据交错路中匹配边与非匹配边的交替性质，由于所有连接 V∖V′V\setminus V'V∖V′ 与 V′V'V′ 的边均关联奇点 Vodd′V'_{odd}Vodd′​ 且均为非匹配边，路径 PPP 只能通过非匹配边从外部进入 Vodd′V'_{odd}Vodd′​。进入 TTT 后，为了保持交替，路径必须沿着交错树边走向 Veven′V'_{even}Veven′​。然而，Veven′V'_{even}Veven′​ 中的点（除 rrr 外）在 TTT 内的唯一匹配边均与其对应的奇点相连，且没有连接到树外的匹配边。这导致 PPP 无法在不重复经过顶点的情况下离开 V′V'V′ 或在 V′V'V′ 内终结。

因此，增广路 PPP 中的所有顶点和边必然完全不属于 V′V'V′，即 V(P)∩V′=∅V(P) \cap V' = \emptysetV(P)∩V′=∅。

因此，对 PPP 进行增广对以 rrr 为根的交错树 TTT 毫无影响。在增广后的新匹配 M′M'M′ 下，交错树 TTT 的匹配状态与结构保持不变，自然也就不能新增以 rrr 为根的增广路。结论得证。

完美匹配的存在性
